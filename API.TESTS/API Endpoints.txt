API Endpoints:

1. User Registration:
   - Endpoint: /signup
   - Method: POST
   - Request Body:
     - full_name (string)
     - email (string)
     - password (string)
     - role (string)

2. User Login:
   - Endpoint: /signin
   - Method: POST
   - Request Body:
     - email (string)
     - password (string)

3. Get User Profile:
   - Endpoint: /profile/:userId
   - Method: GET
   - Parameters:
     - userId (number)

4. Admin Update User Information:
   - Endpoint: /AdminUpdate/:userId
   - Method: post
   - Parameters:
     - userId (number)
   - Request Body:
     - full_name (string)
     - email (string)
     - role (string)

5. User Update User information:
   - Endpoint: /UserUpdate/:userId
   - Method: post
   - Parameters:
     - userId (number)
   - Request Body:
     - full_name (string)
     - email (string)

6. Delete User:
   - Endpoint: /deleteUser/:userId
   - Method: post
   - Parameters:
     - userId (number)
